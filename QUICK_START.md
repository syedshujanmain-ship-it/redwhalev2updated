## 🚀 3-Step Setup (2 Minutes)

### 1️⃣ Get Free API Key
Visit: **https://aistudio.google.com/apikey**
- Sign in with Google
- Click "Create API Key"
- Copy the key

### 2️⃣ Add to Red Whale V1
- Click **⚙️ Settings** (top-right)
- Click **"Add New API Key"**
- Paste your key
- Click **"Add Key"**

### 3️⃣ Start Chatting!
Done! Your bot is ready.

## 🆘 Troubleshooting

### "No API Keys Available" Error?
→ Add at least one API key in Settings (⚙️)

### "All Keys Exhausted" Error?
→ Wait 24 hours OR add more keys

### Keys not working?
→ Check key starts with "AIzaSy"
