# Quick Start Guide - Custom API Keys

## 🚀 Getting Started in 3 Steps

### Step 1: Access Settings
Look for the **Settings icon (⚙️)** in the top-right corner of the chat page, next to the theme toggle button.

Click it to open the API Settings page.

### Step 2: Add Your API Key
1. Click the **"Add New API Key"** button
2. (Optional) Enter a label like "My Personal Key" or "Work Key"
3. Paste your Gemini API key (get it from https://aistudio.google.com/apikey)
4. Click **"Add Key"**

That's it! Your key is now active and will be used for all requests.

### Step 3: Choose Your Model (Optional)
1. Select your preferred Gemini model from the dropdown
2. Click **"Save Model Selection"**

Available models:
- **Gemini 2.0 Flash (Experimental)** - Latest and greatest
- **Gemini 2.5 Flash** - Default, balanced performance
- **Gemini 1.5 Pro** - Most capable
- **Gemini 1.5 Flash** - Fast and efficient
- And more...

## 📊 How It Works

### Default Setup (No Custom Keys)
```
6 Default Keys → 120 Requests/Day → Auto-Rotation
```

### With Custom Keys
```
Your Keys (Priority) → Default Keys (Fallback) → Unlimited Potential
```

### Key Rotation Example
```
Request 1-20:   Custom Key #1
Request 21-40:  Custom Key #2
Request 41-60:  Custom Key #3
Request 61+:    Default Keys (automatic fallback)
```

## 🔑 Getting Gemini API Keys

### Free API Keys from Google
1. Visit: https://aistudio.google.com/apikey
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the key (starts with "AIzaSy...")
5. Add it to Red Whale V1

### Free Quota
- Each free API key: **20 requests per day**
- Add multiple keys: **20 × N requests per day**
- Example: 5 keys = 100 requests/day

## ❓ FAQ

### Q: Are my API keys safe?
**A:** Yes! Keys are stored only in your browser's localStorage and never sent anywhere except Google's Gemini API.

### Q: Can I use any Gemini model?
**A:** Yes! Choose from 7 different models including experimental versions.

### Q: What happens if my keys run out?
**A:** The app automatically switches to default keys. You'll never be stuck.

### Q: Do custom keys affect unrestricted features?
**A:** No! All features remain 100% unrestricted regardless of API source.

### Q: How many keys can I add?
**A:** Unlimited! Add as many as you want.

### Q: Can I remove keys later?
**A:** Yes! Click the trash icon next to any key to remove it instantly.
