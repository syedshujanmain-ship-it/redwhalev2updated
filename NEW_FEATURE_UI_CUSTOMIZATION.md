## 🎯 How to Use

### Step 1: Open Customization
Click the **purple paintbrush icon (🎨)** in the header.

### Step 2: Choose a Tab
- **Icons** - Customize header buttons
- **Colors** - Change color scheme
- **Layout** - Adjust spacing

### Step 3: Make Changes
- Use sliders to adjust sizes
- Use color pickers to change colors
- Toggle switches to show/hide icons
- Edit text fields to rename labels

### Step 4: Done!
Changes save automatically. Click **Back** to return to chat.

---

## 💡 Quick Examples

### Hide Unused Icons
1. Open UI Customization
2. Go to Icons tab
3. Toggle OFF icons you don't use
4. Done! They're hidden.

### Make Icons Bigger
1. Open UI Customization
2. Go to Icons tab
3. Move Size slider to 5
4. Done! Icons are larger.

### Change Colors
1. Open UI Customization
2. Go to Colors tab
3. Pick new colors
4. Done! App uses new colors.
